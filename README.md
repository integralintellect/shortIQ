# ShortIQ - URL Shortener

## 🚀 Features

* Shorten long URLs
* Redirect to original URL
* Fast and simple UI

## 🛠 Tech Stack

* Backend: Java (Spring Boot)
* Frontend: JavaScript (React JS)

## ▶️ How to Run

### Backend

cd backend
run application

### Frontend

cd frontend
npm install
npm start

## 📌 Future Improvements

* Add authentication
* Analytics (click tracking)
* Custom short URLs
